Impressive. Let's do some more bean counting.
We will make a vector of length 100.
The first number is 1.
The second number is (1 + 1)^2 = 4.
The third number is sqrt(4 + 4), which is roughly 2.828427.
The fourth number is (2.828427 + 2.828427)^2 = 32.
The fifth number is sqrt(32 + 32)...
Do you see the pattern?
The question is: Of how many digits does the final number consist?
