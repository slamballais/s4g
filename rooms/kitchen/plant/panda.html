Time to do some archery.
Each round consists of 3 arrows.
You can score by being as close to the center as possible:
* ≤ 10 cm: 100 points
* ≤ 20 cm: 80 points
* ≤ 30 cm: 60 points
* ≤ 40 cm: 40 points
* > 40 cm: no points

Load the file `archery_match.rds`.
It contains a list, where each element is a player.
Each player gets to play 30 rounds, so each element contains a list of length 30.
Each of THOSE elements contains a list of three vectors, one per arrow.
Each vector contains the X and Y coordinates of where the arrow landed (0,0 being the center of the board).
So, to grab the X-Y coordinates of the 2nd arrow of the 18th round of player 7, we do archery[[7]][[18]][[2]].
For each round, calculate the scores per arrow, and sum them together.
For each player, find the round with the highest score.
To get the hidden word, sum up all the highest scores (1 per player).
